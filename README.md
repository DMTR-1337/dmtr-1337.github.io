# ParrotOS Terminal Portfolio

An interactive **ParrotOS-style terminal portfolio** with a draggable terminal window. Explore my work by typing commands like a real terminal!

---

## Features

- Draggable terminal window
- Interactive commands:
  - `help` – List commands
  - `about` – About me
  - `skills` – My technical skills
  - `achievements` – Achievements
  - `socials` – Socials
  - `projects` – Projects
  - `neofetch` – Realistic neofetch
  - `whoami` – Displays username
  - `su` – Switch username
  - `clear` – Clear terminal
